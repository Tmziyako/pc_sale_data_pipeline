
# 🖥️ PC Sales Data Pipeline

## 📌 Project Overview

This project demonstrates an end-to-end **data pipeline** designed to process PC sales data from raw ingestion through staging and transformation, and load it into a **star schema data warehouse** for analytics and reporting.

# 🏗️ Architecture
The pipeline follows a layered architecture:
#**Raw Layer:** Source CSV files
#**Staging Layer:** Cleans and transforms data using stored procedures
#**Analytics Layer:** Star schema for reporting
